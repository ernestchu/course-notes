# My course notes
Built with [vuepress](https://v2.vuepress.vuejs.org)

Visit the [page](https://ernestchu.github.io/course-notes/)

### Development
Install dependencies

```
yarn
```
Start a development server

```
yarn docs:dev
```
### Notes
